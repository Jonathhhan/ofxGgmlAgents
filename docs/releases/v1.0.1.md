# ofxGgmlAgents v1.0.1

## Compatibility

- openFrameworks: `of_v20260406_vs_64_release`
- minimum `ofxGgmlCore`: `1.0.1` / commit `2171e8b`
- `OFXGGML_AGENTS_VERSION_STRING`: `1.0.1`

## Highlights

- Added independent Agents addon version metadata.
- Exposed version metadata through `ofxGgmlAgents.h`.
- Documented the release checklist and release policy for agent workflows.
- Added a quickstart for clone, validation, planner example generation, and
  local LLM handoff.
- Kept agent-specific request helpers out of `ofxGgmlCore`.
- Preserved the root-level `ofxGgmlAgentsPlannerExample` as the first example
  lane.

## Runtime Scope

This release includes public request/result helper types, input validation,
description helpers, a root-level planner example skeleton, local validation,
and headless tests.

It does not include complete planner loops, tool registries, local model
runners, memory handoff, RAG integration, or model-backed agent runtimes yet.
Those remain roadmap tasks.
